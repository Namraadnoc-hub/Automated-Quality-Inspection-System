# Automated Bolt Quality Inspection System
## Overview

This project was developed as part of the DecodeLabs Industrial Training Program.
The system automatically detects an industrial bolt using Computer Vision techniques and determines whether it passes quality inspection.

---

## Technologies Used

- Python
- OpenCV
- NumPy

---

## Features

- Image Loading
- Grayscale Conversion
- Gaussian Blur
- Binary Thresholding
- Contour Detection
- Bounding Box Detection
- PASS / FAIL Decision
- Output Image Generation
